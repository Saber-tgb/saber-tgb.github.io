mockServer的思路

  • 以 json-server 作为 mock 服务器， mock.js 生成 mock 数据，以此达到 mock-server 要求。

安装

  • 全局安装npm install -g json-server
  • 项目安装npm install --save-dev json-server
  • 安装mock.js快速创建模拟接口 npm install --save-dev mock.js

配置

  • 在项目根目录下创建mock文件夹
